# 🎨 Smart Color Palette Extractor

A PHP-based web application that extracts the dominant colors from an uploaded image and generates a beautiful color palette instantly.

## 🚀 Live Demo

**Website:** https://colorextractor.freedev.app

## ✨ Features

* Upload images in multiple formats
* Extract the top 10 dominant colors
* Display additional extracted colors
* Copy HEX color codes with one click
* Responsive and modern UI
* Image preview before extraction

## 🛠️ Tech Stack

* PHP
* HTML5
* CSS3
* JavaScript
* Composer
* League Color Extractor

## 📦 Installation

1. Clone the repository

```bash
git clone https://github.com/missRimjhim777/color-extractor-project.git
```

2. Navigate to the project folder

```bash
cd color-extractor-project
```

3. Install dependencies

```bash
composer install
```

4. Start a PHP server

```bash
php -S localhost:8000
```

5. Open your browser

```
http://localhost:8000
```

## 💡 How It Works

1. Upload an image.
2. The application stores the uploaded image.
3. The League Color Extractor library analyzes the image.
4. The top dominant colors are extracted.
5. The colors are displayed as cards with their HEX values.
6. Users can copy any HEX code with a single click.
